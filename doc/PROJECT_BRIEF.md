# 프로젝트 개요 (Project Brief)

## 한 줄 요약
**1조원 이상 유가증권 운용사를 위한 K-IFRS 기반 헤지회계 자동화 시스템 프로토타입**

---

## 프로젝트 정보

| 항목 | 내용 |
|---|---|
| **프로젝트명** | 헤지회계 자동화 시스템 프로토타입 |
| **목적** | 수주용 PoC (Proof of Concept) |
| **기간** | 4주 |
| **단계** | 프로토타입 개발 |
| **타겟 고객** | 1조원 이상 원화/외화 유가증권 운용사 |

---

## 배경

### 현재 문제점
1조원 이상 운용사의 회계팀은 K-IFRS 1109호 기반 헤지회계 업무에서 다음 문제를 겪고 있음:

- **시간**: 월말 마감 시 3~5일 소요 (엑셀 수작업)
- **오류 위험**: 수작업으로 인한 분식회계 리스크
- **담당자 의존**: 퇴사/휴가 시 업무 마비
- **감사 대응**: 매 감사 시 자료 재작성 반복
- **규제 변경**: K-IFRS 개정 시 수작업 업데이트

### 기회
- AI + 자동화 기술 성숙
- 금융권 디지털 전환 가속화
- 감사 규제 강화 → 체계적 시스템 필요성 증대

---

## 목표

### 정량 목표
- 월말 마감 시간: **3일 → 30분** (95% 감소)
- 계산 오류: **제로화**
- 감사 대응 시간: **1주 → 1일**

### 정성 목표
- K-IFRS 1109호 완벽 준수
- 감사 추적성 100% 보장
- 규제 변경 신속 대응
- 담당자 의존성 해소

---

## 범위

### ✅ 포함 (In-Scope)
- 공정가치 위험회피 회계처리
- 현금흐름 위험회피 회계처리
- 헤지 유효성 테스트 자동화
- 회계 분개 자동 생성
- K-IFRS 근거 자동 인용
- 기본 대시보드 및 리포트

### ❌ 제외 (Out-of-Scope)
- 해외사업장순투자 위험회피
- 금융상품 손상(ECL)
- IFRS 17 보험계약
- 실시간 Bloomberg 연동 (본 개발 시)
- ERP 실제 연동 (본 개발 시)
- 대용량 처리 최적화 (본 개발 시)

## 핵심 기능 (5개)

| # | 기능 | 설명 |
|---|---|---|
| 1 | **헤지 지정 & 문서화** | K-IFRS 6.4.1 요건 자동 검증, 문서 자동 생성 |
| 2 | **공정가치 자동 평가** | Level 1/2/3 분류, 평가방법 자동 적용 |
| 3 | **유효성 테스트 자동화** | Dollar-offset, 회귀분석 |
| 4 | **회계 분개 자동 생성** | 공정가치/현금흐름 헤지 분개 + K-IFRS 근거 |
| 5 | **리포트 출력** | 내부관리/공시/감사 대응 자료 |
