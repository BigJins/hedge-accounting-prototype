## 2. 평가 방법론: IRP 공식 및 Day Count Convention

### 2.1 이자율 평형 이론(IRP) 적용 근거

통화선도의 공정가치는 이자율 평형 이론(Interest Rate Parity, IRP)을 적용하여 산출합니다. IRP는 두 통화 간 금리 차이가 선물환율에 반영된다는 이론으로, 시장에서 실제로 통화선도 가격 결정에 사용되는 표준 방법론입니다.

K-IFRS 1113호 문단 61~66은 Level 2 공정가치 측정에 있어 시장참여자가 자산·부채의 가격을 결정할 때 사용하는 가격결정기법과 투입변수를 사용해야 함을 요구합니다. IRP 모형은 국내외 은행 및 증권사가 통화선도 거래 시 공통으로 사용하는 시장 표준 가격결정기법입니다.

**핵심 공식**

```
현재 선물환율 = S₀ × (1 + r_KRW × T/365) / (1 + r_USD × T/360)

현가계수(Discount Factor) = 1 / (1 + r_KRW × T/365)

통화선도 공정가치(KRW) = (현재 선물환율 - 계약 선물환율) × 명목원금(USD) × 현가계수
```

| 변수 | 의미 | DEMO 값 |
|---|---|---|
| S₀ | 평가기준일 현물환율 (KRW/USD) | 1,350.00 |
| r_KRW | 원화 무위험이자율 (국고채 기준, 소수) | 0.035000 (= 3.5%) |
| r_USD | 달러 무위험이자율 (SOFR 기준, 소수) | 0.053000 (= 5.3%) |
| T | 잔존일수 (평가기준일~만기일) | 92일 |
| 계약 선물환율 | 계약 체결일에 확정된 환율 | 1,380.00 KRW/USD |
| 명목원금 | 계약 USD 금액 | USD 10,000,000 |

### 2.2 Day Count Convention (일수 계산 관행)

통화선도 IRP 계산에는 통화별로 상이한 일수 계산 관행(Day Count Convention)을 적용합니다. 이는 각국 금융시장의 역사적 관행에서 비롯된 국제 표준입니다.

| 통화 | 관행 | 분모 | 근거 |
|---|---|---|---|
| **KRW** | Actual/365 Fixed | 365 (윤년에도 고정) | 한국 CD금리·국고채·통안채 자금시장 표준 |
| **USD** | Actual/360 | 360 (윤년에도 고정) | USD SOFR·구LIBOR Money Market 국제 표준 |

**Actual/365 Fixed vs. Actual/Actual 구분**

Actual/Actual(윤년 적용, 분모가 실제 연간 일수)은 국고채 장기물 쿠폰 계산에만 해당합니다. 통화선도 IRP 계산에서는 윤년 발생 시에도 KRW 분모를 365로 고정하는 Actual/365 Fixed를 적용합니다. 이는 단기 자금시장의 국제적 표준이며, K-IFRS 1113호 문단 61~66에서 요구하는 "시장참여자가 사용하는 가격결정기법 적용" 원칙에 부합합니다.

**K-IFRS 1109호 예시 주석 37의 인정**

K-IFRS 1109호 예시 주석 37은 KRW와 USD의 Day Count Convention이 다름(365 vs. 360)을 헤지 비효과성(hedge ineffectiveness)의 잠재적 원인으로 명시적으로 인정합니다. 이는 Day Count Convention 차이가 회계상 허용된 불가피한 시장 관행임을 확인합니다.

## 3. DEMO 수치 기반 계산 예시

### 3.1 전제 조건

가나금융투자 박지영 과장 케이스를 기준으로 합니다.

| 항목 | 값 |
|---|---|
| 계약번호 | FX-2024-001 |
| 명목원금 | USD 10,000,000 |
| 계약 선물환율 | 1,380.00 KRW/USD |
| 계약일 / 헤지 지정일 | 2024-01-15 |
| 만기일 | 2024-07-15 |
| 평가기준일 | 2024-03-31 |
| 현물환율 (S₀) | 1,350.00 KRW/USD |
| 원화이자율 (r_KRW) | 3.50% (0.035000) |
| 달러이자율 (r_USD) | 5.30% (0.053000) |
| 잔존일수 (T) | 92일 (2024-03-31 ~ 2024-07-15) |

### 3.2 단계별 계산

**Step 1: KRW 이자 계수 산출 (Actual/365 Fixed)**

```
krwFactor = 1 + r_KRW × T/365
          = 1 + 0.035 × 92/365
          = 1 + 0.035 × 0.252055
          = 1 + 0.008822
          = 1.008822
```

**Step 2: USD 이자 계수 산출 (Actual/360)**

```
usdFactor = 1 + r_USD × T/360
          = 1 + 0.053 × 92/360
          = 1 + 0.053 × 0.255556
          = 1 + 0.013544
          = 1.013544
```

> USD는 Actual/360 관행 적용으로 분모가 360입니다. Actual/365를 잘못 적용하면 usdFactor = 1.013361로 계산되어 약 450원/USD 이상 오차가 발생합니다.

**Step 3: 현재 선물환율 산출**

```
현재 선물환율 = S₀ × krwFactor / usdFactor
             = 1,350.00 × 1.008822 / 1.013544
             = 1,361.9097 / 1.013544
             = 1,343.7098 KRW/USD  (소수점 4자리 반올림)
```

**Step 4: 현가계수 산출**

```
현가계수 = 1 / krwFactor
         = 1 / 1.008822
         = 0.991252  (소수점 6자리 반올림)
```

**Step 5: 공정가치 산출**

```
공정가치 = (현재 선물환율 - 계약 선물환율) × 명목원금 × 현가계수
         = (1,343.7098 - 1,380.00) × 10,000,000 × 0.991252
         = (-36.2902) × 10,000,000 × 0.991252
         = -362,902,000 × 0.991252
         = -359,728,422 원  (소수점 2자리 반올림)
```

### 3.3 결과 해석

| 항목 | 값 | 해석 |
|---|---|---|
| 현재 선물환율 | 1,343.7098 KRW/USD | 시장 현재 선물환율이 계약 시점보다 낮음 (달러 약세) |
| 공정가치 | **-359,728,422 원** | 평가손실 — 위험회피수단이 불리한 포지션 |
| 공정가치 변동액 | -359,728,422 원 | 최초 평가이므로 변동액 = 공정가치 전액 |
| 회계처리 | 평가손 당기손익 인식 | K-IFRS 1109호 6.5.8: 위험회피수단 공정가치 변동 → P&L |

**평가손 발생 원인**: 계약 체결 시점(2024-01-15)의 현물환율이 1,380원대에서 2024-03-31 기준 1,350원대로 달러가 약세를 보임. USD 선도매도(KRW 수취) 포지션을 보유한 상황에서 달러 약세는 불리하게 작용합니다.

**헤지 효과**: 위험회피대상항목인 USD 예금은 달러 약세로 원화 환산 가치가 감소하므로 공정가치 하락 손실이 발생합니다. 통화선도의 평가손(-359백만 원)은 USD 예금의 외화환산 손실을 상쇄하여 헤지 목적을 달성합니다.

## 4. Level 2 분류 근거

### 4.1 분류 기준

K-IFRS 1113호 문단 72~90은 공정가치 측정에 사용된 투입변수에 따라 공정가치 수준을 3단계로 분류합니다.

| 수준 | 정의 | 통화선도 해당 여부 |
|---|---|---|
| Level 1 | 동일 자산·부채에 대한 활성시장의 공시가격(무조정) | 해당 없음 (통화선도는 거래소 상장 상품이 아님) |
| **Level 2** | **직접 또는 간접으로 관측가능한 투입변수** | **해당 (현물환율, 이자율 모두 시장 관측가능)** |
| Level 3 | 관측불가능한 투입변수 | 해당 없음 |

### 4.2 통화선도 Level 2 판정 근거

USD/KRW 통화선도 평가에 사용되는 투입변수:

| 투입변수 | 관측가능 여부 | 출처 | Level 판정 |
|---|---|---|---|
| 현물환율 (S₀, KRW/USD) | 관측가능 | 서울외국환중개, 한국은행 | Level 2 근거 |
| 원화 무위험이자율 (r_KRW) | 관측가능 | 국고채 시장수익률, 한국은행 기준금리 | Level 2 근거 |
| 달러 무위험이자율 (r_USD) | 관측가능 | SOFR (Fed 발표), Bloomberg | Level 2 근거 |

모든 투입변수가 시장에서 관측가능하므로 K-IFRS 1113호 문단 89(관측가능 투입변수 우선 원칙)에 따라 Level 2로 고정 분류합니다.

### 4.3 Level 1이 아닌 이유

통화선도는 장외파생상품(OTC)으로 거래소에 상장되지 않습니다. 따라서 "동일 자산·부채의 활성시장 공시가격"인 Level 1 요건을 충족하지 못합니다. 그러나 IRP 공식의 투입변수(현물환율, 이자율)는 모두 시장에서 관측가능하므로 Level 2가 적절합니다. 이는 IRS(이자율스왑), CRS(통화이자율스왑) 등 대부분의 장외파생상품에 공통으로 적용되는 분류입니다.

## 5. 감사 대응 Q&A

### Q1. USD이자율 산정 시 왜 분모를 360으로 사용하는가? 한국 기준이면 365가 맞지 않는가?

**A.** 이는 USD 달러 자금시장의 국제적 관행(Actual/360)을 적용한 결과입니다.

원화(KRW)는 국고채·CD금리 등 한국 자금시장 표준인 Actual/365 Fixed를 적용합니다. 반면 달러(USD)는 SOFR(Secured Overnight Financing Rate) 및 과거 LIBOR 계산에 사용되던 Actual/360이 국제 표준입니다.

K-IFRS 1113호 문단 61~66은 시장참여자가 자산 가격을 결정할 때 실제로 사용하는 가격결정기법과 투입변수를 적용하도록 요구합니다. 국내외 금융기관 및 시장참여자가 USD/KRW 통화선도 가격을 결정할 때 KRW에는 Actual/365, USD에는 Actual/360을 사용하므로 이 관행을 따르는 것이 K-IFRS에 부합합니다.

또한 K-IFRS 1109호 예시 주석 37은 두 통화의 일수 계산방법 차이를 헤지 비효과성의 허용된 원인으로 명시적으로 인정하고 있어, 이 적용의 회계적 타당성을 뒷받침합니다.

---

### Q2. 공정가치 평가에 단순이자 방식을 사용하는 근거는 무엇인가? 복리 방식과의 차이는?

**A.** 단순이자 방식은 1년 이하 단기 통화선도 계약의 업계 표준입니다.

IRP 공식에서 `(1 + r × T/base)` 형태의 단순이자 방식은 단기(1년 이하) 금융상품 가격결정의 국제 표준입니다. 연속복리 방식 `e^(rT)` 또는 복리 방식 `(1 + r)^(T/365)`과 비교할 때, 1년 이하 구간에서 결과 차이는 매우 미미합니다.

예를 들어, 잔존 92일 기준 3.5% 이자율에서의 차이:
- 단순이자: 1 + 0.035 × 92/365 = 1.008822
- 복리: (1.035)^(92/365) = 1.008784 (차이: 0.000038, 약 0.004%)

K-IFRS 1113호 문단 61~66에서 요구하는 "시장참여자의 가격결정기법"으로서 은행 간 통화선도 가격은 단순이자 방식으로 고시되므로, 단순이자 방식 적용이 가장 시장에 부합하는 방법론입니다.

---

### Q3. 동일한 계약·기준일에 대한 재평가 요청 시 어떻게 처리되는가?

**A.** PoC 단계에서는 동일 계약 + 동일 평가기준일 조합의 재요청 시 항상 새 계산을 수행하고 기존 이력을 갱신합니다.

이는 시장 데이터(환율, 이자율) 변경 또는 입력 오류 수정 시 재계산 결과를 즉시 반영할 수 있도록 의도된 PoC 동작 방식입니다. 또한 동일 contractId 재제출 시 계약 정보(명목원금, 선물환율, 만기일 등)도 함께 갱신됩니다.

본 개발(운영 시스템)에서는 다음 정책으로 전환을 권장합니다:
- 평가 이력 불변(Immutable) 보존 — 수정 불가
- 재계산이 필요한 경우 별도 정정 절차(감사 근거 첨부) 적용
- K-IFRS 1109호 B6.4.12에 따른 지속적 유효성 평가 이력 보존

---

### Q4. (추가) 전기 공정가치가 0인 경우는 어떤 상황인가?

**A.** 최초 평가 시에는 전기 공정가치가 0으로 설정됩니다.

K-IFRS 1109호에 따르면 통화선도 계약 체결 시점(계약일)의 공정가치는 원칙적으로 0입니다. 계약 체결 시 매수자와 매도자 간 정상거래에서 공정한 시장 조건으로 계약하므로 초기 순포지션은 0이 됩니다. 따라서 첫 번째 평가 시 `previousFairValue = 0`으로 설정하는 것은 K-IFRS의 원칙에 부합합니다.
